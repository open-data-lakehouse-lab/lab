# Project Roadmap

The Open Data Lakehouse Lab follows a milestone-based roadmap.

## Milestones

| Milestone | Name | Description | Status |
| --------- | ---- | ----------- | ------ |
| M0 | Foundation | Project governance, principles, and initial repository structure. | In Progress |
| M1 | Weather Ingestion MVP | Ingestion of real public weather open data into a local lab environment. | To Do |
| M2 | Weather Lakehouse MVP | End-to-end lakehouse implementation (Bronze/Silver/Gold) for weather data. | To Do |
| M3 | AWS Local Replication | Replicating the local lab environment using AWS-like local tools. | To Do |
| M4 | GCP Local Replication | Replicating the local lab environment using GCP-like local tools. | To Do |
| M5 | Analytics API and Web Portal | Building a public-facing API and web portal for data visualization. | To Do |
| M6 | Advanced Operations | Orchestration, quality, observability, and advanced CI/CD. | To Do |
