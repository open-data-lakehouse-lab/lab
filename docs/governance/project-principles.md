# Project Principles

The Open Data Lakehouse Lab is guided by the following core principles:

| Principle | Description |
| --------- | ----------- |
| **Neutrality** | The project is independent and does not represent any corporate interest. |
| **Transparency** | All decisions, code, and data sources are public. |
| **Reproducibility** | Anyone should be able to clone the lab and run it locally. |
| **Open Data** | Only public, open datasets are used. No private data allowed. |
| **Multi-Cloud** | Architecture and tools should be agnostic or easily portable across major clouds. |
| **Modularity** | Components are decoupled to allow for experimentation with different technologies. |
| **Documentation-First** | All features and decisions must be clearly documented. |
