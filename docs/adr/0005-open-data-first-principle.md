# ADR 0005: Open Data First Principle

## Status

Accepted

## Context

To ensure the project remains public and reproducible by anyone without needing proprietary data, we need a data strategy.

## Decision

The project will only use public open datasets. The MVP will specifically use real public weather open data.

No private, corporate, or client data will ever be used or included in the project.

## Consequences

- The project remains safe for public consumption and contribution.
- Anyone can reproduce the results using the same public sources.
- Compliance with open-source and data privacy standards.
