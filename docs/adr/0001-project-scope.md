# ADR 0001: Project Scope

## Status

Accepted

## Context

We need to define the boundaries and primary purpose of the Open Data Lakehouse Lab.

## Decision

The project is a personal open-source laboratory. It focuses on building a multi-cloud open data lakehouse using public datasets.

The project must remain independent and neutral. No corporate context, internal projects, or proprietary systems should be mentioned.

## Consequences

- The project can be shared publicly without any conflict of interest.
- It serves as a personal learning and demonstration tool.
- All dependencies must be open-source or publicly accessible.
