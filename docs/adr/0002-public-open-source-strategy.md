# ADR 0002: Public Open Source Strategy

## Status

Accepted

## Context

The project is intended for the public domain and needs a clear strategy for its open-source nature.

## Decision

The project will be hosted on GitHub under the `open-data-lakehouse-lab` organization. All code, documentation, and issues will be public by default.

We will use the MIT License for code and Markdown for documentation to ensure maximum accessibility and contribution ease.

## Consequences

- No secrets or private information can be stored in the repositories.
- Contributions from the community are encouraged.
- The project's progress is transparent to anyone.
